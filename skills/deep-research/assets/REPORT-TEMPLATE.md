# Research Report Template

Use this structure for research output. Adapt sections as needed — not every report needs every section.

```markdown
## Research: [Topic]

### Summary

[2-3 sentence overview of key findings and overall confidence level]

### Key Findings

1. **[Finding]** (Confidence: Strong/Moderate/Weak)
   - [Specific evidence with inline citations]
   - [Supporting or qualifying details]

2. **[Finding]** (Confidence: Strong/Moderate/Weak)
   - [Evidence]

### Conflicts & Open Questions

- **[Area of disagreement]**: [Source A] claims X [1], while [Source B] claims Y [2]. [Analysis of why they differ — different contexts, outdated info, different definitions, etc.]

### Gaps & Limitations

- [What couldn't be verified or found]
- [Areas where evidence is thin]
- [Scope limitations of this research]

### References

[1] [Source name] — [URL], accessed [date]
[2] [Source name] — [URL], accessed [date]
```

## Adaptation Notes

- **Quick check**: Summary + Key Findings + References is sufficient
- **Standard review**: Add Conflicts & Gaps sections
- **Deep-dive**: Add all sections; consider adding a Methodology section describing search approach
- **Comparative**: Replace Key Findings with a comparison table using consistent criteria
- **Fact-check**: Lead with the claim being checked, then evidence for/against
